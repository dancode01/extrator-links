# 🔗 Extrator de Links

Hub com acesso rápido a ferramentas de extração de links para as principais redes sociais e plataformas de conteúdo.

## Plataformas disponíveis

- **TikTok** — vídeos sem marca d'água
- **YouTube** — vídeos e áudio
- **Instagram** — fotos, vídeos, reels e stories
- **Scribd** — documentos
- **Pinterest** — imagens e vídeos

## Sobre

Este projeto apenas organiza links para ferramentas de terceiros já existentes — nenhum conteúdo é hospedado ou processado aqui.

Desenvolvido por [DanTech](https://github.com/dancode01).
